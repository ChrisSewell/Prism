# @prism/web

Frontend for Prism. Built with Vite + React + TypeScript + Tailwind + shadcn/ui.

## Development

```bash
# From the monorepo root
npm install
cd apps/web
npm run dev
```

The dev server runs on `http://localhost:8080` with a proxy to the signaling server on port 3000.

## Build

```bash
npm run build
```

## Test

```bash
npm test
```
